## Product 1: Mettrico - Restaurant Metrics

**Problem:** Restaurant owners can't track KPIs easily  
**Solution:** Simple dashboard with core metrics  
**Timeline:** MVP in 5 days → $5K MRR in 3 months  
**Moat:** Speed + integration lock-in  
**Outcome:** Sold to competitor for 5× revenue  

**70/30 Application:**
- Used Supabase auth (didn't build custom)
- Stock charts (didn't perfect viz)
- Shipped daily iterations

## Common Success Patterns

**Timing:**
- Week 1: Generate with AI (60-70%)
- Week 2-3: Polish with Cursor (70-80%)
- Week 4: Beta users (feedback)
- Month 2: Iterate to 90%
- Month 3-6: Reach 100%

**Moat Building:**
- All built moats WHILE shipping
- Speed → data/network/brand
- None waited to be "ready"

**Key Lessons:**
✅ Ship imperfect, iterate fast  
✅ Users tell you what matters  
✅ AI can generate 60-70% of code  
✅ Real feedback beats planning  
✅ Build moat while building product
